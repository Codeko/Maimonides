Guía de creación de nuevo año escolar de Maimónides

Maimónides trabaja con multitud de datos que provienen de diferentes orígenes esta guía estructura y documenta cada uno de los pasos a seguir para configurar un nuevo año escolar en Maimónides.

Creación del año escolar

El primer paso para configurar un nuevo año escolar es, lógicamente, crearlo. Es este año escolar se asociarán todos los datos. Los años escolares son totalmente independientes unos de otros pero en ciertos informes con carácter interanual se usarán los datos de todos los años escolares.

Para crear un año escolar vaya al menú “Datos”, apartado “Años escolares”. Pulse el botón “Nuevo”. Se creará un nuevo año escolar con inicio en el año actual. Puede cambiar tanto el año como la descripción de este. Una vez empiece a trabajar con un año escolar no debe cambiar el año de este ya que puede producir incongruencias en los datos a la hora de trabajar con Séneca.

Una vez creado el año escolar márquelo y pulse el botón “Activar”. Con esto marcaremos ese año como el año en curso y trabajaremos con él a partir de ahora.

Importación de datos

Existe una gran variedad de datos a importar desde Séneca, algunos datos no se pueden importar si no se han importado antes otros. Por ejemplo si importa el listado de alumnos antes de haber importado los cursos y unidades la importación no se realizará correctamente. Antes de importar los datos al programa debe haberlos rellenado en Séneca.

El orden de importación recomendado es:

Archivo de datos generales

Este archivo lo proporciona Séneca para los distintos generadores de horarios, proporciona información Profesores, Cursos/Grupos, Materias, Dependencias y Tramos Horarios. Es por lo tanto el primer archivo a importar.

Puede descargar este archivo e importarlo o dejar que Maimónides lo haga automáticamente.

Para descargarlo manualmente:

En Séneca en el desplegable superior izquierdo:

  • “Utilidades”->”Intercambio de información”.
  • Desplegable “Sentido del intercambio”-> “Exportar datos desde Séneca”
  • Desplegable “Tipo de intercambio de información”->”Exportación hacia generadores de horarios”.
  • En la esquina superior derecha pulsar el botón añadir.
  • En la pantalla que saldrá pulsar aceptar.
  • Aparecerá el archivo “ExportacionHorarios.xml”.

Una vez importados los datos puede que quiera revisar los siguientes aspectos:

  • Emails de profesores nuevos.
  • Abreviaturas de asignaturas.
  • Número de faltas para la perdida de evaluación continua de las asignaturas.
  • Fechas de inicio y fin de los diferentes cursos.
  • Máximo de faltas para la perdida de evaluación continua en los diferentes cursos.
  • Cambiar o añadir descripciones a las dependencias.

Archivo de tutores

Este archivo permite especificar el tutor de cada grupo. Puede asignar manualmente los tutores, descargar el archivo desde Séneca o dejar que Maimónides importe automáticamente el archivo.

Si desea descargarlo manualmente siga estas instrucciones:

En Séneca en el desplegable superior izquierdo:

  • “Alumnado”->”Unidades”->”Relacion de unidades”.
  • En el desplegable de curso seleccionar “Cualquiera”.
  • Pulsar el botón “Exportar datos”.
  • En el desplegable “Exportar datos al formato” seleccionar “Fichero CSV”.
  • Pulsar "Aceptar" aparecerá para descargar el archivo RegUnidades.csv.

Una vez importado se deben haber asignado correctamente los tutores de cada unidad. Es recomendable revisar que esta asignación se haya hecho correctamente.

Datos del alumnado

Este archivo permite especificar los diferentes alumnos y sus datos extendidos. Puede asignar manualmente el fichero de datos o dejar que Maimónides lo importe automáticamente.

Si desea descargarlo manualmente siga las siguientes intrucciones:

En Séneca en el desplegable superior izquierdo:

  • “Alumnado”->”Alumnado del centro”.
  • En la pantalla que saldrá pulsar aceptar sin modificar ningún filtro.
  • Pulsar el botón “Exportar datos”.
  • En el desplegable “Exportar datos al formato” seleccionar “Hoja de Microsoft Excel”.
  • Pulsar aceptar aparecerá para descargar el archivo RegAlumn.xls.

Código séneca del alumnado

Este fichero permite recuperar el código Séneca de cada alumno, información que no viene en el fichero de datos extendidos.

Como siempre puede elegir asignar el fichero a mano o que maimónides lo recupere automáticamente.

Para descargarlo manualmente debe seguir las siguientes instrucciones:

En Séneca en el desplegable superior izquierdo:

  • “Utilidades”->”Intercambio de información”.
  • Desplegable “Sentido del intercambio”-> “Exportar datos desde Séneca”
  • Desplegable “Tipo de intercambio de información”->”Exportación faltas alumnado”. No importa que no haya faltas. Este fichero se usa para recuperar los datos de alumnos.
  • En la esquina superior derecha pulsar el botón añadir.
  • Añadir por cada curso todas las unidades. Para ello en el desplegable “Curso” seleccionar el curso. En la parte de abajo aparecera una lista “Unidades candidatas”. Pulsar el botón y las unidades se moveran a la lista “Unidades seleccionadas”. Pulsar ahora el botón “Añadir” y se copiarán a recuadro inferior “Seleccion”. Seleccionar otro curso en el desplegable de curso y repetir la operación con todos los cursos.
  • Una vez añadidas todas las unidades de todos los cursos pulsar el botón aceptar
  • La web volverá sola a la página de “Intercambios de información”.
  • Volver a seleccionar Desplegable “Sentido del intercambio”-> “Exportar datos desde Séneca”.
  • Desplegable “Tipo de intercambio de información”-> “Exporatción faltas Alumnado”.
  • Aparecerá debajo una lista con los ficheros generador. Hacer clic en la fecha del último y seleccionar del menú que aparecerá la opción “Descargar documento”.
  • Descargar el archivo “Exportacion_Faltas_Alumnado.zip” y guardarlo.

El proceso de asignación automática puede ser lento y es recomendable verificar que se han asignado correctamente todos los códigos ya que séneca puede dar error en muchos casos, además la recuperación automática sólo debe usarse cuando se recuperan datos del año en curso ya que debido a un error en Séneca este proporciona siempre el último código aunque se le asigne un año anterior.

Datos de matriculación

Los datos de matriculación contienen el estado de las matrículas de cada alumno. Puede descargar los diferentes ficheros de datos e importarlos en el programa o dejar que Maimónides lo haga automáticamente.

Si desea descargar los ficheros sigua los siguientes pasos:

En Séneca en el desplegable superior izquierdo:

  1. ”Alumnado”-> “Matriculación”->”Relación de matriculas”.
  2. En el desplegable “Curso“ seleccionar el primer curso.
  3. Pulsar el botón “ Materias de las que se ha matriculado cada alumno” en la esquina superior derecha.
  4. En la nueva pantalla pulsar el botón “Exportar datos” en la parte superior de la pantalla.
  5. En el desplegable “Exportar los datos al formato” seleccionar “Ficheo CSV” y pulsar aceptar en la esquina superior derecha.
  6. Aparecerá para descargar el fichero “MatOMCMatr.csv”. Guardarlo con el nombre que se desee conservando la extensión csv.
  7. Despues de esto la web volverá automaticamente a la pantalla de “ Materias de las que se ha matriculado cada alumno”.
  8. Seleccionar el siguiente curso y repetir la operación desde el punto 3.
  9. NOTA: Si se utiliza Firefox para descargar los ficheros estos se renombrarán automaticamente al descargarlos. Si se usa internet explorer habrá que cambairle el nombre manualmente.
  10. Desde el importador puede seleccionar todos los archivos para que se importen y no tener que importar archivo por archivo.

Si importa los datos automáticamente desde Séneca debe tener en cuenta que debe estar trabajando en el año actual o si es un año diferente no debe haber habido cambios en los cursos.

Es interesante revisar que se hayan importado correctamente las matriculaciones (viendo que haya datos en el punto de matriculaciones) pues séneca tiene tendencia a devolver ficheros excel erroneos.

Importación de datos de convivencia

En este punto se importan los diferentes datos necesarios para rellenar los partes de convivencia compatibles con séneca. Este apartado sólo permite su carga desde Seéneca directamente ya que no existe ningún punto en Séneca para exportarlos manualmente.

Es recomendable revisar que se han importado correctamente desde el punto de configuración de convivencia.

Importación de horarios

Actualmente maimónides es capaz de importar horarios desde Horw. Para que la importación sea correcta los nombres de profesores, materias, cursos, dependencias, etc deben ser coincidentes con séneca.

Importación de calendario escolar

Desde este punto podrá importar el calendario escolar del centro desde Séneca. Para más información visite la ayuda del Panel de Calendario.

Acciones posteriores a la importación

Una vez se hayan importado los datos es necesario realizar una serie de tareas. Entre ellas:

  • Asignar las divisiones de horarios en el apartado de horarios. Howr no proporciona información de división de alumnos en los diferentes casos en los que varias unidades den clase conjuntamente en un aula o se divida una unidad en varioas aulas. Esta información ha de especificarse manualmente.
