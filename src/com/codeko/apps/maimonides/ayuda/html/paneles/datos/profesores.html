Profesores

Acceso rápido Crtl + P

Descripción

Desde esta pantalla, si es usted un usuario autorizado, podrá consultar el listado de profesores, crear uno nuevo, borrar uno ya existente, o si dispone de una web compatible con Maimónides, enviarlos a la web.

  • Actualizar: Permite cargar el listado de profesores de manera inicial, o de nuevo si se ha producido algún cambio para verificar que los cambios se hayan aplicado con éxito.
  • Borrar: Para borrar un profesor selecionelo del listado y pulse el boton Borrar, Maimónides le solicitará confirmación antes de proceder al borrado. Puede borrar varios profesores de una sola vez realizando una selección múltiple (pulsando la tecla "Control" mientras realiza las selecciones).
  • Nuevo: Permite introducir un nuevo profesor en Maimónides, tenga en cuenta que esto no lo introducirá en Séneca.
    Pulsando el boton nuevo se creará una nueva fila editable en la tabla, donde podrá introducir cada campo los datos correspondientes.
    Para finalizar de introducir un profesor pulse la tecla "Enter" o haga click fuera del campo.
  • Actualizar Datos: Permite descargar desde Séneca los datos de los profesores.
    Haciendo click sobre el botón "Actualizar Datos" Maimónides le presentará un ventana de dialogo, si usted desea únicamente importar datos nuevos y no modificar los existentes marque la casilla habilitada a tal efecto en el cuadro de diálogo.
    Mas adelante, en la misma ventana de diálogo podrá escoger entre importar estos datos directamente desde Séneca o bien desde un fichero descargado previamente de Séneca.
    • Importar datos desde Séneca: Maimónides le solicitará que ingrese su usuario y Clave antes de proceder a la importación, una vez introducidos esta se realizará de manera automática
    • Importar datos desde Fichero: Maimónides abrirá una ventana para que pueda seleciona en su ordenador el fichero previamente descargado
      El fichero desde el que esos datos son importados es el Archivo de Datos Generales, maimónides únicamente importará de este archivo los datos relativos al profesorado.
      Haga click aqui para recibir ayuda acerca de cómo obtener este archivo.
  • Enviar a la web: Permite, si su centro poseé una web compatible con Maimónides correctamente configurada enviar los datos de profesores a su web.
    Tenga presente que sólo seran enviados aquellos profesores que tengan asociado un e-mail, ya que el e-mail es el usuario de acceso a la web compatible.
  • Horarios Seleccionados: Permite imprimir los horarios de los profesores que se encuentren seleccionados, tanto si es uno o varios.
  • Horarios: Permite imprimir los horarios de todos los profesores del listado.
  • Si desea editar un profesor haga doble click sobre el campo que desea editar, este campo pasará as er editable en ese momento. Para finalizar la edición pulse la tecla "Enter" o haga click fuera de ese campo.

Cada profesor se muestra en una fila de la tabla y consta de varios campos. Maimónides sólo muestra por defecto los campos de uso mas habitual, si usted desea desplegar o replegar los campos mostrados haga uso del boton ubicado en las esquina superior derecha de la tabla identificado con el icono de una tabla y una pequeña flecha.
Una vez pulsado este botón usted podrá activar o desactivar los campos mostrados.
A continuacion se detalla en que consiste cada uno.

  • Código Séneca: Es el código mediante el que se identifica a ese profesor en Séneca. Es posible que algunos profesores, no posean ese código, bien por que no esten registrados en Séneca bien por que no dependan de la Administración.
  • Nombre, 1er Apellido, 2º Apellido: El nombre y apellidos del profesor.
  • Puesto: El puesto que ocupa ese profesor en el centro, usualmente se muestra la materia que imparte.
  • F. Toma: La fecha de toma, es la fecha en la que el profesor comenzó a trabajar en ese puesto dentro del centro.
  • E-mail: El e-mail del profesor, Maimónides hace uso de él para en envío de notificaciones y el envío a una web compatible, siendo el e-mail el nombre de usuario de acceso a la web.
