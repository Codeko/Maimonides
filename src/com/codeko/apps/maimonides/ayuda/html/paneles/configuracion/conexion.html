Configuración de conexión

Acceso rápido Crtl + C

Descripción

Desde este punto usted podra configurar la conexión del programa con el servidor. Habitualmente no es necesario puesto que nuestros técnicos lo habrán realizado.
En caso se ser necesario debe completar los siguientes parámetros.
Dirección de la base de datos: La dirección IP del servidor.
Nombre de la base de datos: El nombre de la base de datos.
Usuario de acceso a la base de datos: Usualmente suele ser root.
Clave de acceso a la base de datos: La contraeña que identifica al usuario en la base de datos.

Le rogamos encarecidamente que no modifique estos parámetros. En caso de que no se establezca la conexión consulte con nuestros técnicos.

Ruta: Archivo -> Configuración -> Configuración de conexión.
