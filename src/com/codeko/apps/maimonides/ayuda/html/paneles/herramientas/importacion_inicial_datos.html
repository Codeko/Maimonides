Importación Inicial de Datos

Descripción

Cada vez que se comienza un nuevo año escolar es necesario cargar en Maimónides una serie de datos que le permitirán comenzar a trabajar, estos datos se extraen desde Séneca y desde su generador de horarios.
Este panel le permite importar estos datos.

Para mas información acerca de esta pantalla puede consultar el siguiente enlace: Guía de creación de nuevo año escolar de Maimónides.

Ruta: Herramientas -> Importación inicial de datos.
