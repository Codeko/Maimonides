Copias de Seguridad

Acceso rápido Crtl + Alt + B

Descripción

Esta pantalla sólo esta visible para el Administrador. Si usted es Administrador podrá realizar una nueva copia de seguridad, restaurar una copia ya hecha, o consultar las copias ya realizadas.
Así mismo se muestra información de cuando fué realizada la última copia de seguridad.

Esta pantalla va a desaparecer en nuevas versiones, automatizando la copia de seguridad.

Ruta: Herramientas -> Copias de Seguridad
