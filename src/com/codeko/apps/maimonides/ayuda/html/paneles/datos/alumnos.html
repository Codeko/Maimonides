Alumnos

Acceso rápido Crtl + F

Descripción

Desde este punto, si es usted un usuario autorizado, podrá consultar la ficha del alumno, viendo de una sola ojeada que porcentaje le resta al alumno para perder la evaluación continua, datos de asistencia, horarios, comunicaciones, etc.. Permite asimismo gestionar la convivencia, asignar apoyos, o agregar un nuevo alumno.

El panel de almuno se subdivide en tres pestañas, para consultar la ayuda relativa a cada pestaña haga click en el nombre

  • Ficha
  • Listado
  • Convivencia

Ruta: Datos -> Alumnos
