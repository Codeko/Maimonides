Materias

Acceso rápido Crtl + T

Descripción

Desde este punto usted podrá consultar las materias ofertadas en el centro, eliminar las que desee, y definir si son evaluables. Asimismo podrá decidir cual es el máximo de faltas permitidas para esa materia.
La pantalla de Materias se divide en dos partes

Menú superior

El menú superior consta de una botonera con las diferentes acciones que se pueden llevar a cabo.

  • Actualizar: Permite actualizar la lista de materias. Actulice si se estan editando las materioas en varios puestos de manera simultánea
  • Nuevo: Aunque el listado de materias se obtiene por Maimónides de manera automática desde el programa Séneca, maimónides permite que se inserten nuevas materias o se modifiquen las existentes. para añadir una nueva materia haga click sobre el botón "Nuevo", Maimonides crearçá un nuevo campo al final de la tabla dónde podrá completar los datos realtivos a dicha materia.
  • Borrar: El boton "Borrar" permite eliminar una materia, para eliminarla selecionela con el ratón y haga click en borrar. Para evitar borrados accidentales Maimónides le mostrará una venta de confirmación antes de proceder al borrado.

Zona de Datos

La zona central y mas amplia es la Zona de Datos en ella se muestra de manera tabulada los datos relativos a cada materia.
Si desea editar algún dato únicamente debe hacer docble click sobre el dato que desea modificar, ese campo se convertirá en editable de manera automática, para finalizar la edicón pulse "Enter" o haga click fuera de ese campo.
Cada materia tiene asociados una serie de datos, Maimónides sólo muestra por defecto los campos de uso mas habitual, si usted desea desplegar o replegar los campos mostrados haga uso del boton ubicado en las esquina superior derecha de la tabla identificado con el icono de una tabla y una pequeña flecha.
Una vez pulsado este botón usted podrá activar o desactivar los campos mostrados.

  • Código: Indica el código en Séneca de dicha asignatura. Este dato extrae de Séneca y puede presentar problemas si se modifica.
  • Curso: El curso al que corresponde dicha materia. Este dato se extrae de Séneca y puede presentar problemas si se modifica.
  • Abreviatura: Se asigna de manera automática por Maimónides, es editable haciendo doble click sobre dicha abreviatura.
  • Descripción: La descripción de la materia, este dato se extrae de manera automática desde Séneca.
  • Evaluable: Determina si la asignatura es o no evaluable, este dato se asigna por maimónides en funcion de los siguientes parámetros, si dicha asignatura tiene código en Séneca se marca como evaluable, si la asignatura no tiene código en Séneca se marca como no evaluable. Este dato se puede modificar haciendo click sobre la casilla, aquen puede presentar problemas si usted marca como no evaluable una materia que si conste como tal en Séneca.
  • Capacidad: Determina el número de alumnos máximos en dicha asignatura. Este dato se extrae de Séneca de manera automática y puede presentar problemas si se modifica.
  • Max. Faltas: Define el máximo de faltas que se pueden acumular en dicha asignatura antes de perder la evaluación continua. Este dato se debe introducir manualmente, el número indica el total de faltas acumulables antes de perder la evaluación continúa.
