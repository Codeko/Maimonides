Dependencias

Acceso rápido Crtl + D

Descripción

Desde esta pantalla usted podrá consultar las dependencias del centro, y si dipone de los permisos de usuario necesarios asignar una nueva dependencia o eliminar una ya existente.
La pantalla se encuentra dividia en dos partes

Menú superior

El menú spuerior se compone de una botonera con las diferentes acciones que pueden llevarse a cabo

  • Actualizar: Actualiza la lista de dependencias disponibles y sus datos. Si se está editando en varios puestos de manera simultanea utilice "Actualizar", esto le mostrará el listado de dependencias diponibles.
  • Nuevo: Si desea añadir una nueva dependencia pulse el boton Nuevo, Maimónides le solicitará que introduzca un nombre para la dependencia y una descripción.
    Para escribir sobre un campo basta con que haga doble click donde desea escribir.
  • Borrar: Si desea eliminar una dependencia, seleccione con un click la dependencia que desea eliminar y pulse el boton Borrar. maimónides le solicitará confirmación antes de proceder al Borrado. Puede borrar varios profesores de una sola vez realizando una selección múltiple (pulsando la tecla "Control" mientras realiza las selecciones).

Zona de Datos

La zona que ocupa la mayor parte de la pantalla es la Zona de Datos, en ella se muestran las dependencias.
Cada dependencia esta identificada con un Nombre Descripción y un Código que puede consultar haciendo click sobre el icono situado en la parte superior derecha de la tabla
El código se asigna de manera automática por Maimónides, mientras que Nombre y Descripcion se deben indicar manualmente.
Si desea editar una dependencia haga doble click sobre el campo que desea editar, este campo pasará as er editable en ese momento. Para finalizar la edición pulse la tecla "Enter" o haga click fuera de ese campo.
