Notificaciones de Convivencia

Descripción

Desde este punto usted puede gestionar de manera automática la comunicacion con los padres / tutores y el envío automático a Séneca relativo a la convivencia.

  • Expulsiones
    Permite enviar a los padres /tutores las notificaciones relativas a la convivencia mediante diferentes medios, carta, e-mail y/o SMS.
    Mustra una pantalla con las notificaciones permitiendo filtrar los datos por varios aspectos, curso, grupo, fecha, o si ya han sido notificadas o estan pendientes
    Haciendo click sobre el nombre del alumno se muestra la ficha de dicho alumno.
  • Amonestaciones
    Permite enviar las amonestaciones a Séneca de manera automática y desatendida, la pantalla permite filtrar los datos por diferentes campos, tales como curso, grupo, tipo de conducta contraria, si ha sido o no revisado pr el tutor etc...
