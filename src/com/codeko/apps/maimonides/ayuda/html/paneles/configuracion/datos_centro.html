Configuración de Datos del Centro

Descripción

Desde este punto usted podra configurar los datos relativos a su centro:

  • Código:
    Es el código Séneca que identifica a su centro.
  • Nombre:
    Es el nombre de su centro. Este dato se usará, entre otras cosas, para la generacion de documentos.
  • Dirección, población, código postal y provincia:
    La dirección postal del centro. Este dato se usará, entre otras cosas, para la generacion de documentos.
  • Télefono y fax
    El teléfono y/o fax del centro. Este dato se usará, entre otras cosas, para la generacion de documentos.
  • E-mail:
    El e-mail del centro, este dato se usará como remitente para el envio de notificaciones.
  • Web compatible:
    En el caso de utilizar una web compatible con maimónides marque la casilla e incluya el usuario y clave para permitir a Maimónides el envío de datos a la web

Ruta: Archivo -> Configuración -> Configuración general -> Datos del centro.
